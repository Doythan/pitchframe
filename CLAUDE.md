# PitchFrame

## 프로젝트 목표
PitchFrame은 조축/친구들끼리 쉽게 포메이션을 만들고,
그 결과물을 프로 전술 카드처럼 고급스럽게 보여주는 서비스다.

## MVP 범위
- 포메이션 프리셋: 4-4-2, 4-3-3, 4-2-3-1, 3-5-2
- 축구장 위 선수 마커 렌더링
- 선수 이름 수정
- 드래그로 위치 조정
- 팀명 / 전술 제목 / 짧은 설명
- 테마 선택
- 실시간 미리보기
- PNG export
- localStorage 저장/복원

## 제품 원칙
- 편집은 쉽게
- 결과물은 프로답게
- 기능 수보다 결과물 퀄리티 우선

## v1 비범위
- 백엔드 서버 없음
- Spring Boot 없음
- 로그인 없음
- 데이터베이스 없음
- 협업 없음
- AI 기능 없음
- 관리자 기능 없음
- 분석 기능 없음

## 기술 스택
- Next.js App Router
- TypeScript
- Tailwind CSS
- Zustand
- html-to-image

## UI 방향
- 전술 보드 느낌
- 스포츠 분석 카드 느낌
- 여백과 정렬이 깔끔해야 함
- 일반적인 SaaS 대시보드 느낌은 피할 것
- 오른쪽 미리보기 결과물이 서비스의 핵심

## 엔지니어링 원칙
- 먼저 파일을 확인하고 판단할 것
- 오버엔지니어링 금지
- 불필요한 추상화 금지
- 유지보수 가능한 단순한 구현 선호
- 임시 파일은 마지막에 정리
- 의미 있는 변경 후 검증 수행

## 완료 조건
사용자가 다음을 할 수 있어야 한다.
- 포메이션 선택
- 선수 이름 수정
- 선수 위치 드래그 이동
- 팀명/전술 정보 수정
- 결과 미리보기 확인
- PNG export
- 새로고침 후 마지막 상태 유지
